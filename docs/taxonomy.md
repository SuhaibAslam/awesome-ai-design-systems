# AI-Native Design System Taxonomy

## AI-Driven Component Generation
Systems that generate UI components from prompts, schemas, or intent.

Examples: 21st.dev

## AI Chat / Agent UI Systems
Primitives for chat, agents, tool-calling, streaming responses, and handoffs.

Examples: OpenAI ChatKit, assistant-ui, ChatUI

## AI UI Component Libraries
Component libraries focused on AI interaction patterns and assistant UIs.

Examples: Lobe UI, LangUI, Ant Design X

## Copilot / Assistant Integration Systems
Frameworks for embedding copilots into existing applications.

Examples: CopilotKit

## Prompt-First / Prompt-Component Systems
Treat prompts as composable, reusable, versionable UI primitives.

Examples: mdx-prompt, mui-prompt

## Schema-to-UI Kits
Render structured AI outputs directly into UI components.

Examples: AINativeKit, Autoview

## Agent/UI Protocols
Protocols and schemas for agent-to-UI interactions.

Examples: AG-UI Protocol
